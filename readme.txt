== Description ==

= What this plugin does: =

Automatically displays a "reddit" button for each post and page.

= Note: This plugin is soon to be discontinued. =

* Check out our new social media button plugin: [S-ButtonZ (Social Media Buttons Plugin)](http://wordpress.org/extend/plugins/s-buttonz/)

= Features: =

* Full admin options for reddit buttons. 

* Choose to display reddit button at top left or right or bottom left or right. 

* Can hide buttons on home page, pages, posts, tag pages, archives and search results page. 

* Hide buttons on a single page or post [See Faq](http://wordpress.org/extend/plugins/reddz-et/faq/).

== Installation ==

1. Upload the "reddZ-Et" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Click "Setting" tab and find a link to "reddZ-Et" options
4. Fill out the options

== Frequently Asked Questions ==

= Which version of WordPress is supported? =

WordPress 2.5.x to 2.7.x is currently supported. This plugin has not been tested on any versions before 2.5.

= How do I hide a button on a single post or page? =

 Include the following code in your post or page: 

 &lt;!--reddZ=none--&gt;
